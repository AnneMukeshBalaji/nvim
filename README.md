To execute this repository, you need to install Packer.nvim, a plugin manager for Neovim.

Installation Instructions

Windows :

git clone --depth 1 https://github.com/wbthomason/packer.nvim.git $env:USERPROFILE\AppData\Local\nvim\pack\packer\start\packer.nvim

macos :
git clone --depth 1 https://github.com/wbthomason/packer.nvim.git ~/.local/share/nvim/site/pack/packer/start/packer.nvim 

linux :
git clone --depth 1 https://github.com/wbthomason/packer.nvim.git ~/.local/share/nvim/site/pack/packer/start/packer.nvim
